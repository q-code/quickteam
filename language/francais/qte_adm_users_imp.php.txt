Le fichier CSV doit contenir: role, login, mot de passe, email. Les roles sont : A (administrateur), M (membre du staff), U (utilisateur). Pour le mot de passe vous pouvez utiliser : le code sha1, le mode passe avec le pr&eacute;fixe SHA, ou laisser le mot de passe vide pour obtenir un mot de passe &eacute;gal au login.<br /><b><i>Exemple de fichier csv:</i></b><br />M, john, 5753a498f025464d72e088a9d5d6e8725, john@corp.com<br />
U, william, SHAwilly03, william@corp.com<br />
U, robert, , robert@corp.com
